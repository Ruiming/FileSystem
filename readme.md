# 资源管理器
  基于Angular, Node.js, Electron的Windows资源管理器

### 截图预览
![image](https://raw.githubusercontent.com/Ruiming/FileSystem/master/static/windows.png)

### 主要功能：

  - 文件和文件夹的复制粘贴新建删除
  - 搜索功能
  - 显示系统信息
  - 监控文件和文件夹的操作
  - 文件分类，用户和权限管理
